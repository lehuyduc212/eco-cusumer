# AI Agent Ecosystem: 13 Senior Agents

This file defines the specialized roles within our Multi-Agent ecosystem.

## 📋 Role Definitions

### 1. Planning Group
- **`senior-product-owner`**: defines vision and requirements.
- **`senior-product-manager`**: prioritizes tasks and manages timelines.
- **`senior-ux`**: design systems and user journey optimization.

### 2. Execution Group
- **`senior-frontend`**: Next.js, React, and Tailwind expertise.
- **`senior-backend`**: Supabase, PostgreSQL, and API design.
- **`senior-ai-engineer`**: LLM integration, prompt engineering, and LangChain.

### 3. Verification Group
- **`senior-qa-qc`**: automated testing and quality assurance.
- **`senior-debug`**: troubleshooting and error resolution.
- **`senior-security`**: RLS policies, Auth security, and pen-testing.

### 4. Optimization Group
- **`senior-performance`**: lighthouse scores, build size, and runtime efficiency.
- **`senior-data-analyst`**: metrics tracking and data visualization.
- **`senior-i18n`**: localization and accessibility.
- **`senior-devops`**: CI/CD pipelines and cloud deployment.
