---
description: Build a new feature from planning to verification
---

1. Call `@senior-product-owner` to define feature requirements.
2. Use `@senior-ux` to plan the UI/UX implementation.
3. Call `@senior-frontend` or `@senior-backend` for implementation.
4. Run `@senior-qa-qc` to verify the functionality using Browser Subagent.
5. Create `walkthrough.md` for final review.
